================================================================================
  C examples for LibTiePie

  LibTiePie is a cross platform library for controlling TiePie engineering USB
  oscilloscopes and arbitrary waveform generators

  See https://www.tiepie.com/nl/libtiepie-hw-sdk for information about supported instruments,
  platforms, SDK documentation, etc.

  (c) 2014-2023 TiePie engineering
================================================================================

The examples show how oscilloscopes and generators can be controlled using LibTiePie in C.


Requirements:

- These C examples, from https://www.tiepie.com/libtiepie-hw-sdk/c .

- Windows:
  LibTiePie library from https://www.tiepie.com/libtiepie-hw-sdk/windows .
  The libtiepie-hw.dll file and the libtiepie-hw.h header file must be located in
  the main folder of the examples.

- Linux:
  Install LibTiePie library from https://www.tiepie.com/libtiepie-hw-sdk/linux .

- Compiler/toolchain:
  - GCC.

Building the examples:

- Windows:
  The examples can be built by executing mingw32-make in the main folder of the examples.

- Linux:
  The examples can be built by executing make in the main folder of the examples.
